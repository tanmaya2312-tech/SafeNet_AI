# 🛡️ SafeNet AI

> AI-Powered Cybersecurity Platform

SafeNet AI is a web-based cybersecurity platform built using **Python** and **Streamlit** that helps users detect common cyber threats through AI-powered analysis and interactive security tools.

---

## 🚀 Features

### 🔐 Password Checker
- Evaluates password strength
- Calculates a security score
- Provides improvement suggestions
- Generates downloadable PDF reports

### 🌐 URL Scanner
- Analyzes suspicious URLs
- Identifies potential security risks
- Displays threat analysis

### 📨 Scam Detector
- Detects phishing and scam messages
- Provides security recommendations
- Generates PDF reports

### 🤖 AI Cybersecurity Assistant
- Powered by Sarvam AI
- Answers cybersecurity-related questions
- Beginner-friendly explanations

### 📊 Interactive Dashboard
- Security Overview
- Threat Statistics
- Analysis History
- Overall Security Score
- Daily Cybersecurity Tips

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/tanmaya2312-tech/SafeNet_AI.git
```

Go to the project folder

```bash
cd SafeNet_AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
